# Build and Package AmoraDigitalRock

This document summarizes how to build and package AmoraDigitalRock on Windows.

AmoraDigitalRock developed by João Rafael Da Silveira, José Frank Vanzeler, Celso Rafael de Lima, and Marcus Lucas Do Amaral, PhD students from Federal University of Pará.

This application is part of the Geophysics workspace, a space with distinct applications focused on AI solutions for geoscience. 

The sponsors for this work is SEBRAE, Catalisa ICT. 

This work is done in collaboration with LPSA (Laboratório de Petrossísmica Sustentável da Amazônia), UFPA (Universidade Federal do Pará).
