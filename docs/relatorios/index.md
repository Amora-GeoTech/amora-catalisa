# Como enviar o Relatório Mensal

Opção A (arquivo):
1. Duplique o arquivo `docs/relatorios/AAAA-MM.md` do mês correspondente.
2. Preencha os campos e faça *commit* + *push*.

Opção B (Issue Form):
1. Vá em **Issues → New issue → Relatório Mensal**.
2. Preencha e envie. (Opcional) Copie o conteúdo para um arquivo em `docs/relatorios/`.

> Dica: anexe evidências via links (vídeos, prints, planilhas).
